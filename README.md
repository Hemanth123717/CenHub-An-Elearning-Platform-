CenHub is a comprehensive eLearning platform designed to enhance digital education through structured assessments and programming practice.

Developed for students, educators, and institutions to manage learning and testing activities.
Supports subjects like Aptitude, Verbal Ability, Reasoning, and programming languages (Java, C, C++, Python).
Features a built-in coding environment for real-time code execution.
Employs Role-Based Access Control (RBAC) for secure user management.
Offers personalized dashboards for performance tracking and analytics.

Purpose of CenHub
The primary goal of CenHub is to create a centralized system for efficient access to learning content and assessments.

Aims to provide an engaging platform for academic and competitive exam preparation.
Encourages active learning through structured MCQ tests and coding practice.
Incorporates RBAC for secure, role-specific access to different panels.

Scope of CenHub
CenHub encompasses the design and deployment of a web-based platform for online assessments and coding practice.

Supports multiple subjects and is intended for educational institutions and training centers.
Facilitates three user roles: Client (Student), Admin, and Server (Super Admin).
Designed to be scalable and secure, with future expansion capabilities for additional features.
